##  项目概览

**Nihon Cine Prompt** 是一个面向 GPT Image 2 的交互式提示词词典，系统收录了日本独立电影与新浪潮运动中 8 位导演/摄影师的视觉美学特征，并将其转化为可直接复制使用的英文提示词。从筱田升的逆光柔焦到北野武的冷蓝暴力，从寺山修司的超现实幻彩到园子温的霓虹癫狂——每一种美学都被拆解为可组合的提示词模块。

> 本项目不仅是一个提示词列表，更是一份**关于日本电影摄影美学的微型百科全书**。

##  使用方法

### 方式一：直接使用在线版（推荐）

访问部署地址，搜索/筛选提示词，一键复制英文 prompt 到 GPT Image 2：

> [在线访问地址（请替换为你的 GitHub Pages 地址）](https://你的用户名.github.io/nihon-cine-prompt)

### 方式二：本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/nihon-cine-prompt.git

# 2. 进入目录
cd nihon-cine-prompt

# 3. 安装依赖
npm install

# 4. 启动开发服务器
npm run dev

# 5. 浏览器打开 http://localhost:3000
```

### 方式三：构建静态文件

```bash
npm run build
```

构建完成后，`dist/` 目录中包含完整的静态网站文件，可直接部署到任何静态托管服务。
